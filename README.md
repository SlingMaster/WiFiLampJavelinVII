# WiFiLampJavelinVII

Project WiFi Lamp Javelin VII & Remote Control • III

Нижче посилання на відео:

[![Watch the video](https://img.youtube.com/vi/06ELf2x0_ys/0.jpg)](https://www.youtube.com/watch?v=06ELf2x0_ys)

У файлі `ConstantsUser.h` розкоментуйте потрібний варіант конструкції лампи:

```cpp
// #define ROUND_LAMP     // оригінальна схема лампи «Gyver» (Round Lamp)
// #define ORG_LAMP       // оригінальна схема лампи «Gyver»
// #define JAVELIN        // оригінальна схема лампи «JAVELIN»
#define JAVELIN_VII       // PRO-схема лампи «JAVELIN VII»
```

Універсальна прошивка для трьох конструкцій лампи (`ROUND_LAMP` – різновид лампи Гайвера).

За замовчуванням використовується остання конструкція лампи.

![JavelinVII](Schemes/Icon/ico_javelinVII.png)

## Дії енкодера для прошивки JAVELIN_VII:

- одноразовий клік – увімкнення або вимкнення світильника;
- поворот за годинниковою стрілкою – наступний ефект;
- поворот проти годинникової стрілки – попередній ефект;
- дворазовий клік – зміна режиму меню по циклу: "Set Speed, Set Scale, Effect";
- натиск і поворот – зміна яскравості лампи (Set LED Brightness, крок 10% у діапазоні -50%…+50% від яскравості ефекту за замовчуванням);

## Дії сенсорної кнопки для прошивки JAVELIN_VII:

- одноразовий клік – увімкнення лампи на 5 хвилин (незалежно від того, чи вона вже увімкнена);
- утримання при вимкненій лампі – вмикає ефект «Біле світло»;
- дворазовий клік – команда для лампи «Вогонь»;
- триразовий клік – motion control (переводить лампу в режим зміни ефектів від датчика руху);
- чотириразовий клік – cycle effect (переводить лампу в режим циклічної зміни ефектів);
- п’ятиразовий клік – вмикає ефект «Intim» (вогонь з іскрами);
- шестиразовий клік – переводить лампу в режим Developer Mode (на екрані відображаються Free Memory та FPS);
- семиразовий клік – зміна режиму роботи лампи: з Wi-Fi точки доступу на Wi-Fi клієнт або навпаки;

## Круговий індикатор стану лампи JAVELIN_VII:

- зелена секторальна анімація при старті – підключення до роутера;
- синя секторальна анімація при старті – робота в режимі точки доступу;
- 3 сектори по 8 LED за годинниковою стрілкою (швидкість, яскравість, масштаб).
