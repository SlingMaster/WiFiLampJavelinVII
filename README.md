# WiFiLampJavelinVII

## Project WiFi Lamp Javelin VII & Remote Control • III

Firmware для WiFi-ламп «Javelin» та «Javelin VII», сумісна зі схемою gunner47, а також кросплатформне програмне забезпечення для керування лампами.

**Можливості прошивки Version 7.XX (135 effects)**

- Прошивка для лампи з оригінальною конструкцією «Javelin», сумісна з gunner47.
- Робота в режимі WiFi точки доступу або як WiFi клієнт для роутера.
- Вбудована HTML-сторінка для першого підключення до WiFi-мережі.
- Автосканування доступних ламп у мережі.
- Можливість тонкого налаштування під свої потреби з урахуванням конструкції лампи:
  - а) конструктивні особливості: файл `ConstantsUser.h`
  - б) функціональні вподобання: файл `button.ino` (закоментовані альтернативи поведінки, які можна налаштувати на свій смак)

- Управління матрицями та стрічками на базі адресних світлодіодів WS2812B, можливі варіанти:
  - LED-стрічка до 1024 LED
  - матриці: 8×8, 8×32, 16×16
  - а також композитні:
    - а) дві матриці 16×16
    - б) три матриці 16×16 по вертикалі
    - в) три матриці 8×32 по горизонталі

- Підтримка кругового індикатора стану та механічної або сенсорної кнопки для оригінальної конструкції лампи «Javelin» або «Javelin VII».
- Робота в режимі WiFi точки доступу або як WiFi клієнт для роутера.
- Зовнішнє управління лампою за допомогою протоколів HTTP, MQTT, UDP.

Формат даних JSON:
```json
{"cmd": number, "val": number, "valStr": string}
```

- Ключі `val` та `valStr` не обов’язкові для більшості команд.
- Коди команд для лампи знаходяться у файлі `Constants.h`.

Інші можливості:

- Рандомне налаштування ефектів.
- Автоматична зміна ефектів.
- Будильник «Світанок» (оновлена, більш реалістична версія).
- Управління групою ламп (до 7 пристроїв) у поєднанні з WiFi Lamp Remote Control.
- Таймер вимкнення при тривалому використанні (за замовчуванням – 3 години).
- Таймер вимкнення через 5 хвилин.
- Підтримка файлових систем SPIFFS та LittleFS (за замовчуванням SPIFFS).
- Двомовний список ефектів (українська, англійська).
- OTA (підтримка безпровідного оновлення).
- Тест LED-матриці для перевірки правильності підключення.
- Режим «Javelin» – імітація пострілу та діагностика.
- Управління лампою через сенсорну або механічну кнопку.
- Сповіщення про вихід нових версій прошивки.

## Для схеми «Javelin VII»

- Підтримка OLED-дисплея 128×64:
  - а) індикація режимів роботи
  - б) відображення назви та номера ефекту
  - в) поточний час при спрацюванні датчика руху
  - г) меню налаштувань ефектів

- Автоматичне визначення часового поясу та синхронізація часу.
- Підтримка датчика руху.
- Оптимізовано механізм передачі списку ефектів для програми керування.
- Підтримка керування лампою через Wi-Fi на різних платформах:
  - Android
  - Windows
  - браузер (iOS, Linux)

### WiFi Lamp Projector
вбудований в «WiFi Lamp Javelin Remote»
1. Для переходу до WiFi Lamp Projector натисніть помаранчовий квадрат у caption програми.
2. Зображення для відтворення повинні знаходитися в папці `UI/ani/`.
   - Щоб відкрити папку, натисніть на логотип WiFi Lamp Projector.
   - Можна додавати власні зображення (drag and drop працює лише з цією папкою).
3. Запуск зображення можливий кількома способами:
   - drag and drop
   - кнопки керування: назад, авто, вперед
4. Натисніть кнопку **Transfer** (працює в режимі старт/стоп).
   - Кнопка **24 bit** перемикає color space: RGB888 (за замовчуванням RGB332 — рекомендовано для кращого FPS).
5. Кнопка **Back** — повернення до Lamp Javelin Remote.
6. Тап по зображенню змінює його масштаб до повного без обрізання.
7. **Developer Tool** — інструмент для підбору solid-кольорів і градієнтів у реальному часі без перепрошивки лампи.


![CpuUsage](Schemes/Icon/icoCpuUsage.png) ### CPU Utilization

1. Додаток «CPU Utilization» для Windows.  
   CPU Utilization — додаток для Windows, який дозволяє візуалізувати на лампі рівень завантаження процесора.

#### Можливості додатка:

- автозапуск при завантаженні Windows;
- 4 типи візуалізації;
- тестовий режим;
- тимчасове припинення трансляції.


![ColorMusic](Schemes/Icon/icoColorMusic.png) ### Color Music

Додаток для візуалізації звуку (аналог «світломузики»).  
Не потребує встановлення — достатньо запустити `ColorMusic.exe`.

Перед початком роботи:
- вкажіть IP-адресу лампи
- виберіть папку з музикою у форматі MP3

**Можливості додатку:**

1. Відтворення музики (play, stop, next, back), а також програвання всіх треків із вибраної папки.
2. Регулювання гучності (не впливає на візуалізацію).
3. Кнопка логарифмічного спектру (впливає на візуалізацію, рекомендований режим).
4. Кнопка **Mute** — плавне зниження та підвищення гучності (не впливає на візуалізацію).
5. Інтерфейс у приємному ретро-стилі.
6. Вибір режиму візуалізації прямо з додатку.

- Додаток передає на лампу оброблений звук через UDP-протокол.
- Лампа відображає його відповідно до логіки, реалізованої у прошивці.
- За бажанням можна створювати власні ефекти (див. `sound_visualiser.ino`).
- Кнопка **Developer** зарезервована для розробників.

## Дії енкодера для прошивки JAVELIN_VII:

- одноразовий клік – увімкнення або вимкнення світильника;
- поворот за годинниковою стрілкою – наступний ефект;
- поворот проти годинникової стрілки – попередній ефект;
- дворазовий клік – зміна режиму меню по циклу: «Set Speed | Set Scale | Effect»;
- натиск і поворот – зміна яскравості лампи (Set LED Brightness, крок 10% у діапазоні -50%…+50% від яскравості ефекту за замовчуванням);
- триразовий клік –дефолтні значення ефектів;
- чотириразовий клік – cycle effect (переводить лампу в режим циклічної зміни ефектів);

## Дії сенсорної кнопки для прошивки JAVELIN_VII:

- одноразовий клік – увімкнення лампи на 5 хвилин (незалежно від того, чи вона вже увімкнена);
- утримання при вимкненій лампі – вмикає ефект «Біле світло»;
- дворазовий клік – команда для лампи «Вогонь»;
- триразовий клік – motion control (переводить лампу в режим зміни ефектів від датчика руху);
- чотириразовий клік – cycle effect (переводить лампу в режим циклічної зміни ефектів);
- п’ятиразовий клік – вмикає ефект «Intim» (вогонь з іскрами);
- шестиразовий клік – переводить лампу в режим Developer Mode (на екрані відображаються Free Memory та FPS);
- семиразовий клік – зміна режиму роботи лампи: з Wi-Fi точки доступу на Wi-Fi клієнт або навпаки;

## Круговий індикатор стану лампи JAVELIN_VII:

- зелена секторальна анімація при старті – підключення до роутера;
- синя секторальна анімація при старті – робота в режимі точки доступу;
- 3 сектори по 8 LED за годинниковою стрілкою (швидкість, яскравість, масштаб).
